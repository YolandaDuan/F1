F1 Championship
=====================

This is a Inoic 2 app which built for the formula one fans.

## Prerequists
* node@>=7.9.0
* npm@>=4.2.0
* cordova@>=6.5.0

## Run locally

Install npm dependencies:

```
npm install
```

Then run:

```
ionic serve
```
